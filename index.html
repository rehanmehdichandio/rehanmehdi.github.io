Data Cleaning

Data cleaning is the process of correcting errors and inconsistencies in datasets. Using Excel, this involves removing duplicates, filling missing values, correcting data types, and standardizing formats to ensure accuracy and reliability for analysis.

Data Visualization

Data visualization is the practice of representing data through graphical formats. Using Power BI, this involves creating interactive charts, graphs, and dashboards that provide insights and facilitate data-driven decision-making. Power BI's tools make it easy to visualize complex data in a clear, understandable manner.

Python

Data analysis in Python utilizes libraries such as NumPy for numerical computations, pandas for data manipulation, Matplotlib and Seaborn for visualization, and scikit-learn for machine learning to extract insights and build predictive models.

mySQL

Database management with MySQL involves storing, retrieving, and managing data efficiently using structured query language (SQL), ensuring data integrity and enabling complex queries for robust data analysis.

Tableau

Tableau is a powerful tool in data analysis, renowned for its intuitive visual interface and robust analytics capabilities. It enables users to create interactive dashboards and visualizations, facilitating deep insights and informed decision-making from complex datasets.

Git & Github

I use Git to track changes and manage versions of my projects efficiently. It helps me keep a record of all modifications and collaborate seamlessly with teammates. I leverage GitHub to host my Git repositories online, enabling easy access and collaboration on my projects.
